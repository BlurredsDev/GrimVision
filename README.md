# GrimVision

Плагин для майнкрафт серверов, который дает всем постоянное ночное видение. Просто, быстро, без лишнего.

## Что это?

Зашел на сервер - получил ночное видение. Умер - снова получил. Без частиц, без иконки, без настроек. Работает из коробки.

## Зачем?

- Не надо каждому игроку давать команды
- Не нагружает сервер (эффект выдается раз при входе, а не каждую секунду)
- Удобно для PvP серверов или выживания
- Никаких прав доступа и конфигов

## Установка

Закинь `GrimVision.jar` в папку `plugins/` и перезапусти сервер. Всё.

## Работает на

- Spigot / Paper / Purpur
- Minecraft 1.16.5 и выше
- Java 16+

## Как собрать самому

**Windows:**
```bash
COMPILE.bat
```

**Linux/Mac:**
```bash
chmod +x compile.sh
./compile.sh
```

JAR будет в папке `target/`

## Как работает

Плагин ловит два события:
- Игрок зашел → дает ночное видение
- Игрок респавнится → дает ночное видение

Длительность эффекта `Integer.MAX_VALUE` (примерно 3.4 года игрового времени), так что обновлять каждую секунду не надо.
